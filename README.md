# Daily Bread - Nutrition & Meal Planning Application

A comprehensive web application that provides users with daily fitness tips, nutritious recipes, and AI-powered meal recommendations tailored to individual fitness goals.

## Features

- **AI-Powered Meal Recommendations** based on fitness goals
- **User Profile Management** with body metrics tracking
- **Daily Tips & Recipes** for healthy lifestyle
- **Meal Ordering System** for nutritious meal delivery
- **Analytics Dashboard** to track progress

## Tech Stack

- **Frontend**: React + TypeScript + Tailwind CSS
- **Backend**: Python + AWS Lambda
- **Database**: DynamoDB
- **Infrastructure**: AWS (S3, CloudFront, API Gateway, Route53, ACM)
- **IaC**: AWS CDK

## Getting Started

```bash
# Install dependencies
npm run install:all

# Run frontend
cd frontend && npm start

# Deploy to AWS
cd infrastructure && ./deploy.sh
```

See [docs/README.md](docs/README.md) for complete setup instructions.

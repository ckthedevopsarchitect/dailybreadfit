## Infrastructure Components

### Compute
- **AWS Lambda**: Serverless compute for backend logic
- **Lambda Layer**: Shared Python dependencies
- **Memory**: 512MB (auth), 1024MB (meal recs)
- **Timeout**: 30s (auth), 60s (meal recs)

### Storage
- **DynamoDB**: 7 tables with GSIs
- **S3**: 3 buckets (frontend, content, uploads)
- **Billing**: Pay-per-request (DynamoDB)

### Networking
- **CloudFront**: Global CDN, 300+ edge locations
- **API Gateway**: REST API with CORS
- **Route53**: DNS management (optional)

### Security
- **ACM**: Free SSL/TLS certificates
- **IAM**: Role-based access control
- **Cognito**: User pools (optional alternative to JWT)

### Monitoring
- **CloudWatch Logs**: All Lambda logs
- **CloudWatch Metrics**: API, Lambda, DynamoDB metrics
- **X-Ray**: Distributed tracing (optional)
